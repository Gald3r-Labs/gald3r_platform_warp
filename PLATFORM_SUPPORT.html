gald3r - Supported Platforms

The file-based AI-dev framework, packaged for each AI coding tool as its own repo.

This repository is the Cursor + Claude Code unity edition - clone it and run setup_gald3r_project.ps1 to install gald3r for both at once.
Using a different tool? Each platform has its own dedicated repo below - clone the one you use. (This page replaces the old all-in-one dynamic installer.)
