## Headline

gald3r now installs itself — one CLI, every OS. Get the agent and the throne running with a single command.

---

## What's coming

- **`gald3r install agent|throne|all`** — install the gald3r_agent runtime and the gald3r_throne app for your OS (Windows, macOS, Linux), no manual build dance.
- **A real home for gald3r** — a centralized install home (settings, logs, vault) with a global `gald3r` command that works from any directory, plus a USB-portable mode that leaves no trace on the host.
- **Safe self-update** — `gald3r version-check` and `gald3r upgrade` keep you current with an automatic timestamped backup and roll back on any failure. Your tasks, bugs, and plans are never touched.
- **One-command project scaffolding** — `gald3r init` stands up a fresh gald3r project in any folder (seeding your PROJECT.md), and `gald3r update` brings an existing one up to date — both idempotent, both safe.
- **Vault + identity, provisioned for you** — your install inherits org/brand/vault defaults automatically, and never writes a secret into source control.

---

## Why upgrade now

If you've been hand-wiring gald3r into each project, this is the release that does it for you: install once, set up the products, scaffold projects, and stay updated — all from the `gald3r` CLI, identically across every OS.
